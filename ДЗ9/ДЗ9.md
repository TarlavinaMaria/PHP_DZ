## 1. Создание базы данных

```sql
CREATE DATABASE school;
```

### Таблица студентов

```sql
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    age INT,
    grade VARCHAR(10));
```

### Таблица курсов

```sql
CREATE TABLE courses (
    course_id INT AUTO_INCREMENT PRIMARY KEY,
    course_name VARCHAR(100),
    subject INT);
```
